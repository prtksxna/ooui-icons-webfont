Wikimedia Icons Font

Generated using the OOUI icons in the Mediawiki theme
OOjs UI · Icons · Style guide · Request an icon

Usage

  1. Download the entire ooui-icons-webfontrepository into your project folder.
  2. In the <head> of your HTML link to WikimediaIcons.css.
  3. Click on an icon to get its html code
<html>
	<head>
		<link rel="stylesheet" href="path/to/WikimediaIcons.css" />
	</head>
	<body>
		<i class="<%= baseClass ? baseClass + ' ' : '' %><%= classPrefix %>name"></i>
	</body>
</html>
